AutoDetect Recon Suite is an interactive Bash script designed to streamline the Information Gathering and Enumeration phases for CTFs and security audits.
It automates industry-standard tools like Nmap (NSE scripts), Nikto, Gobuster, and WhatWeb, providing a modular workflow where users selectively deploy scans. 
Features include passive subdomain discovery via crt.sh, automated report generation, and ANSI-clean logging for immediate documentation. Built for speed, precision, and efficiency.
